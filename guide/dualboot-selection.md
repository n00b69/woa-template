# Dualboot selection

There are two methods to dualboot Windows. Select the method you would like to use below.

| **Classic method** | **Dualboot Kernelpatcher**
|------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------
| <a href="3-dualboot.md"><img src="https://github.com/n00b69/woa-template/blob/main/guide/images/zsta.png" width="80"></a> | <a href="dbkp.md"><img src="https://github.com/n00b69/woa-template/blob/main/guide/images/zdbkp.png" width="80"></a>
| - Root required | - No root required
| - Dualboot using an app | - Dualboot using the **PLACEHOLDER1**
| - If the UEFI is flashed, you cannot charge your device if it is turned off | - You can charge your device if it is turned off (magnetic case method only)
| [Classic guide](3-dualboot.md) | [DBKP guide](dbkp.md)



REPLACE **PLACEHOLDER1** with:
> If NABU
**volume buttons** or **magnetic case**
> If CEPHEUS / LG
**assistant button**
> If HOTDOG / GUACAMOLE
**alert slider**

> Delete magnetic case info in **LINE 10** if not NABU
