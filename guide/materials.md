## Toggling USB host mode
> [!Warning]
> Disable USB host mode if you use a powered USB hub, as this can irreversibly damage your device.
>
> If you don't use a powered USB hub, enable USB host mode or you will not be able to use any USB devices.

- Enable USB host mode using the the [USB Host Mode Switch tool](https://github.com/n00b69/usbhostmode).
- If USB host mode is already enabled, disable and re-enable it.

## Install Microsoft Office
- Go to [Gravesoft's Office installer page](https://gravesoft.dev/office_c2r_links).
- Download the installer that fits your purposes. Make sure you select `Online x64`, which will install the **ARM64** version.
- Open the `setup.exe` and follow any instructions provided within.
